# Upload-ai

An artificial inteligence based website where the user can upload a video and use the prompt to ask anything. The answer will be given by the gpt using only the transcription of the uploaded video


## Functionalities

- Video upload
- Transcription
- Exemple prompts
- Edit prompt
- Select temperature
